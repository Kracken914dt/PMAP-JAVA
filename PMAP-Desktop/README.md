# PMAP Desktop

Aplicación de escritorio del módulo Gestión de Materias.

## Tecnologías

- Java 17
- Swing
- JDBC puro
- MySQL
- Maven

## Clase principal

- [com.pmap.desktop.view.Main](src/main/java/com/pmap/desktop/view/Main.java)

## Compilación

Este proyecto está preparado para Maven. Si Maven no está instalado en tu equipo, también puedes abrirlo en IntelliJ IDEA o Eclipse e importar el `pom.xml`.

## Funcionalidades

- Crear materias.
- Listar materias en `JTable`.
- Buscar materias.
- Actualizar materias.
- Eliminar materias.
